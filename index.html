Política de Privacidade

Última atualização: 06 de março de 2026

Esta Política de Privacidade descreve como o Academia App coleta, usa e protege as informações dos usuários. Ao usar o aplicativo, você concorda com os termos descritos abaixo.

1. Informações Coletadas

  • Dados de cadastro: nome completo e endereço de e-mail.
  • Credenciais de acesso: senha, armazenada de forma segura no servidor.
  • Medidas corporais: peso, altura e outras medidas inseridas pelo usuário ou personal trainer.
  • Dados de treino: programas de exercícios e circuitos atribuídos pelo personal trainer.
  • Token de notificação push: identificador gerado pelo Firebase Cloud Messaging (FCM).
  • Dados de sessão: armazenados localmente no dispositivo para manter o usuário conectado.

2. Uso das Informações

  • Autenticar e identificar o usuário no aplicativo.
  • Disponibilizar programas de treino e conteúdos personalizados.
  • Registrar e acompanhar o progresso físico do aluno.
  • Enviar notificações push sobre atualizações de treino e avisos do personal trainer.
  • Gerenciar o plano de acesso (Bronze, Prata ou Ouro).

3. Compartilhamento de Dados

Não vendemos nem compartilhamos dados pessoais com terceiros para fins comerciais. Os dados são acessados pelo personal trainer responsável pela conta do aluno. Utilizamos o Firebase Cloud Messaging (Google) exclusivamente para notificações push.

4. Armazenamento e Segurança

Os dados são armazenados em servidor privado com acesso restrito. Senhas são armazenadas de forma criptografada. Dados de sessão ficam no dispositivo e são removidos ao efetuar logout.

5. Retenção de Dados

Os dados são mantidos enquanto a conta estiver ativa. A exclusão da conta pode ser solicitada ao responsável pelo aplicativo.

6. Direitos do Usuário

  • Acessar os dados que temos sobre você.
  • Solicitar correção de dados incorretos.
  • Solicitar a exclusão da sua conta e dados pessoais.
  • Desativar notificações push nas configurações do dispositivo.

7. Público-Alvo

Este aplicativo é destinado exclusivamente a maiores de 18 anos. Não coletamos dados de menores de idade.

8. Contato

Dúvidas ou solicitações: sie.escolasegura@gmail.com

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Mudanças significativas serão comunicadas pelo aplicativo.
